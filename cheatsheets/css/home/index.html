Elements in css declaration structures

    • The selector can be a class that can contain more than one element of the html

    Types

    • Universal
      *
    • Tag Element
      h1 / p / label / div
    • Classes
      .class-name-example
    • ID
      #name-id-example
    • Per Attribute
      [name='samuel']
    • Descendense
      h1 p / #johnwik label / .mymomy .shoes
    • Pseudo-Classes
      xNameOfXClass:hover
